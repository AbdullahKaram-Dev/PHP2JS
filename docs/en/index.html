PHP2JS

Develop easily and without having to allocate additional server resources!

Share PHP-defined variables directly with the JavaScript files you want to use in your system.

This package has streamlined the development of countless projects in the Laravel Framework, providing seamless communication with the JavaScript frontend.

It has arrived to streamline the creation of monolithic projects easily.

Documentation
From the Controller
You will now have access to new methods in the native View class of the Laravel framework. These methods allow you to define, from the controller, what information to make available for use in your JavaScript files.
From Blade
If you prefer doing it from Blade templates, well, then we've got you covered. You'll have access to a variety of directives that you can use as needed.
Specific Data
We understand that there will be times when you only want to share certain data with your JavaScript. That's why we've developed a feature that allows you to discriminate which variables will be used in the Blade template and which ones will be sent to JavaScript.
Global Values
How does the idea of having access to data as if your application's debugger were active from JavaScript sound to you? You see, this is just one of the many data points that are ready to be utilized.
JavaScript Methods
Of course! In addition to having the data available in your JavaScript files, you will also have access to some methods designed for you to control the information according to your preferences.
Security
Believe us, a team of testers has spent months evaluating this solution, and furthermore, a major multinational corporation has sponsored its ongoing improvement. Feel confident in using it.
